Spec my node
============

A basic bdd framework to work with node so that I can learn a bit of javascript and node. 

To Install
==========
    npm Spec_My_Node

To Use
======
    run spec_my_node <file_name>
or
Create specs in ./spec and then:
    run spec_my_node 

Examples
========

See ./spec folder for examples on use
